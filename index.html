Ni Juan Carlos Beaskoetxea naiz

Informatikaren munduan 30 urte baino gehiagoko ibilbidea daukat, batez ere Sistemen Informazio Geografikoetan (GIS) eta web aplikazioen garapenean espezializatuta. QGIS, OpenLayers, GeoServer (Tomcat bidez) eta Laravel bezalako tresnak erabili ohi ditut, baina, horrez gain, Docker, Apache, Nginx eta PostgreSQL/PostGIS bezalako teknologiekin ere lan egiten dut.

Gaur egun, teknikoki jarduteaz gain, formakuntzan dihardut, enpresei zein pertsona interesdunei gai hauek euskaraz (eta, behar izanez gero, gaztelaniaz) erakutsiz: web garapena (Laravel, NodeJS), datu-baseen kudeaketa (PostgreSQL + PostGIS), sistema eragileen administrazioa (Linux) edo datu-analisi aurreratua (Excel, Power BI, etab.). Helburu nagusia formakuntza praktikoa eskaintzea da, adibide errealekin eta ariketa didaktikoekin, ikasleek ikasitakoa berehala aplikatzeko aukera izan dezaten.

Teknologia berrien garapena etengabe aztertzen dut, beti ere ezagutza zabaltzeko eta formakuntza eguneratu bat eskaini ahal izateko. Edozein galdera, lankidetza edo formazio beharrizan izanez gero, jar zaitez lasai harremanetan!
