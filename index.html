Soy Juan Carlos Beaskoetxea

Soy un programador de aplicaciones web con mucha experiencia en los Sistemas de Información Geográfica (GIS). En este sentido trabajo habitualmente con GIS de escritorio como QGIS, con el Framework Laravel, en su última versión (actualmente la 10), con las librerías de Javascrpt Openlayers (versión 7), con el servidor de mapas Geoserver y, por consiguiente, con TomCat.

En el desarrollo de la mayoría de los proyectos de programación web en los que tomo parte se mezclan estas tecnologías mencionadas, pero se combinan con el uso de otras herramientas como Docker para crear entornos de desarrollo personalizados, Apache y Nginx como servidores de aplicaciones web, y PostgreSQL como Sistema Gestor de Bases de datos Relacionale, incluyendo la indispensable extensión de PostGIS usada para trabajar con GIS.
